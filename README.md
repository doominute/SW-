# 🎓 고대로 앉아 (Go-Dae-Ro-An-Ja)

> **"시험기간 갈 곳 없는 고대인들을 위한 문과캠퍼스 공부 명당 소개 & 집중 타이머 서비스"**

`고대로 앉아`는 고려대학교 **문과 캠퍼스** 내에 잘 알려지지 않은 숨은 공부 명당을 추천하고, 학생들이 스마트폰 앱 차단 설정을 통해 방해받지 않고 공부할 수 있도록 만들어줍니다. 사이트 내의 타이머를 이용하면 하루 동안 얼마나 공부했는 지 역시 확인할 수 있습니다. 

---

## 🌟 주요 기능 (Key Features)

### 1. 📍 공부 명당 추천
* **3D 캠퍼스 안내 지도:** 고려대 인문계 캠퍼스 지도를 바탕으로, 공부 명당을 표시합니다.
* **실시간 혼잡도 제공:** 각 명당의실시간 혼잡도(여유, 보통, 혼잡) 시뮬레이션 및 갱신 기능을 지원합니다.
* **명당 상세 데이터:** 콘센트 여부, 소음도/분위기, 운영 시간, 특징 태그, 상세 설명 정보를 한눈에 확인할 수 있습니다.
* **공부 시작 버튼:** 상세 뷰에서 바로 타이머 탭으로 연결되어 바로 학습을 시작할 수 있습니다.

### 2. 🚫 집중 방해 앱 차단 설정 (허용앱 설정)
* **집중 앱 선별:** 공부하는 동안 사용을 허가할 일부 필수 생산성 앱(노션, 줌, 고대포탈 등)만 활성화하고 나머지는 차단 리스트로 보관합니다.
* **집중 모드:** 타이머가 돌기 시작하면 비허용 앱(인스타그램, 유튜브 등)에 접근하는 것을 제한하는 고정 락(Lock) 화면이 활성화됩니다.

### 3. ⏱️ 학습 타이머 & 백색 소음 플레이어
* **서큘러 게이지 타이머:** 초 단위로 차오르는 원형 게이지 애니메이션 스톱워치 탑재.

### 4. 📊 학습 통계 분석 보고서 (마이페이지)
* **학습 등급 & 업적 잠금해제:** 누적 공부 시간에 따라 학위 등급(학사 ➡️ 석사 ➡️ 박사) 및 개별 도전과제 업적을 해금하는 게이미피케이션 탑재.
* **시각 차트 (Chart.js):**
  * **일별 누적 시간(Bar Chart):** 최근 7일 동안 매일 얼마나 공부했는지 막대그래프로 표시.
  * **장소별 선호도(Doughnut Chart):** 어떤 명당에서 가장 오랫동안 공부했는지 비율로 분석.
* **최근 기록 히스토리 테이블:** 완료 성공한 직전 공부 기록 리스트 노출.
